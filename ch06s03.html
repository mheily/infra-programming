Simple conditionals

A "simple" conditional expresses an either/or situation.

Example 6.3. Conditionals in Ruby

if foo == 'bar'
  puts 'hi'
else
  puts 'bye'
end

Do not use 'unless' in Ruby.

Example 6.4. Conditionals in Bash

if [ "$foo" == "bar" ] ; then
  echo 'hi'
else
  echo 'bye'
end

Use double quotes around each term.
