Creating temporary files

Temporary files need to be created in a secure manner.

Example 6.9. Creating temporary files in Ruby

require 'tempfile'
f = Tempfile.new("blah")
f.puts('hi')
f.flush
system("cat #{ f.path }")

The above example will create a temporary file with the first part of the name being "blah". The file will be automatically deleted when the program exits.

Example 6.10. Creating temporary files in Bash

tmpfile=$(mktemp "/tmp/blah.XXXXXXX")
echo 'hi' > $tmpfile
cat $tmpfile
rm $tmpfile

Note that in Bash, temporary files are not automatically deleted when the program exits.
