Running system commands

Sometimes, you need to run some system commands and make sure they complete successfully.

Example 6.11. Running system commands in Ruby

system 'pgrep foo' or raise 'the foo process has died'

Example 6.12. Running system commands in Bash

pgrep foo || ( echo 'the foo process has died' ; exit 1 )
