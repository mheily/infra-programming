Python

Pros:

  • Popular

  • General-purpose programming language

  • Simple grammar

  • Limited syntactic sugar

Cons:

  • Not a pure object-oriented language

  • The standard library doesn't support exec(). You have to 'import os'

  • Dissimilar to Bash

    1. whitespace is significant

    2. mandatory parenthesis when calling functions

  • Missing features

    1. backticks

    2. case statements

    3. the $? variable

    4. string interpolation

    5. first class regular expressions

    6. control flow operators (&& and ||)
