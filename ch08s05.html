Configuration files

You can use YAML to define a configuration file containing simple key-value pairs. Given the following configuration file:

debug: true
server: 'myserver.acme.com'
port: 8080

You could read this into a hash named 'config' like so:

Example 8.5. Reading a configuration file

require 'yaml'

config = YAML.load_file('/etc/my_config_file')
puts config['server'] + ':' + config['port']
