Case statements

A case statement should be used if there are more than two possible conditions.

Example 6.5. Case statements in Ruby

case foo
when 'bar'
  puts 'hi'
when 'baz'
  puts 'ho'
else
  puts 'bye'
end

Example 6.6. Case statements in Bash

case $foo in
bar)
  echo "hi"
  ;;
baz)
  echo "ho"
  ;;
*)
  echo "bye"
  ;;
esac
