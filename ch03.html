Don't Repeat Yourself (DRY)

Here is an example of a program with duplicate logic: 


x = "hello"
x.upcase!
y = "John"
puts x + ' ' + y

x = "goodbye"
x.upcase!
y = "Fred"
puts x + ' ' + y

x = "hello"
x.upcase!
y = "Wanda"
puts x + ' ' + y

To de-duplicate this program, use a function. Example: 

def say(action,person)
  action.upcase!
  puts action + ' ' + person
end

say 'hello', 'John'
say 'goodbye', 'Fred'
say 'hello', 'Wanda'
