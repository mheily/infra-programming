Creating temporary files

Temporary files need to be created in a secure manner.

Example 8.6. Creating temporary files

require 'tempfile'
f = Tempfile.new("blah")
f.puts('hi')
f.flush
system("cat #{ f.path }")

The above example will create a temporary file with the first part of the name being "blah". The file will be automatically deleted when the program exits.
