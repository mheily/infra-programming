Entry point

Some languages have a function called main() that acts as the designated entry point for the program. Ruby does not have this concept, so execution just starts wherever the first statement is declared.

Example 5.1. Entry point

def sub1
  puts 'hi'
end

def sub2
  puts 'bye'
end

# Program execution starts here

sub1
sub2
exit 0
